Using the command prompt

We will be using the command prompt to navigate directories (folders), open Atom, and run python. When you first open up the command prompt, it will start you in your user's Home directory. That's usually something like C:\Users\<username>\.

Learn these simple commands (always lowercase):

  1. ls - lists the contents of the current working directory
  2. cd - change directory to a path relative to your current working directory or an absolute path
     #This is a relative path. .. means go up one.  
     $ cd ..
     #This means go up 2 folders 
     $ cd ../..
     #This means cd into `somefolder` that is in the current directory 
     $ cd somefolder 
     #This is an absolute path 
     $ cd C:\Python27\
     # Worth noting that while in windows you use \ not / for paths. 

  3. mkdir - make a new folder

     # This makes a new folder called newfolder 
     $ mkdir newfolder

  4. rm - this deletes files

     # Deletes a file called somefile 
     $ rm somefile

Using Atom

To write our Python scripts, we will be using Atom. Atom has a lot of nice plugins you can find online, if you're interested. For now, I'll assume you have the basic installation (whatever that means).

$ atom [folder or filename]

You usually want to open Atom in a folder so that you can see the folder manager with all your files in a left-side panel in Atom. You can create, navigate, delete, and duplicate files and folders from that panel.

Running scripts

$ python scriptname.py

Remember, all paths must be relative to your current directory or an absolute path.
